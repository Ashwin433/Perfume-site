Perfume E-commerce Website

A clean, responsive perfume store website built with vanilla JavaScript, HTML, and CSS. No frameworks, no bloat - just straightforward code that works.

Features
Product Showcase - Beautiful image slider with thumbnails
Subscription Options - Toggle between single and double subscriptions
Fragrance Selection - Choose from multiple perfume variants
Responsive Design - Works on mobile, tablet, and desktop
Interactive Elements - Smooth animations and transitions
Tech Stack
Frontend: Vanilla JavaScript, HTML5, CSS3
No Build Tools - Just open index.html in any browser
No Dependencies - Zero external libraries or frameworks
